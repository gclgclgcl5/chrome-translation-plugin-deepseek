智能文本助手 Chrome 插件 - 快速开始指南
================================================

📋 部署步骤（5分钟完成）：

1. 生成图标文件
   - 双击打开 icon-generator.html 文件
   - 点击"生成图标"按钮
   - 右键保存三个图标到 icons 文件夹：
     * icon16.png
     * icon48.png
     * icon128.png

2. 加载插件到Chrome
   - 打开 Chrome 浏览器
   - 访问 chrome://extensions/
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择本文件夹（浏览器插件）

3. 配置API Key
   - 访问 https://platform.deepseek.com 注册账号
   - 创建并复制 API Key
   - 在 Chrome 插件管理页面点击"扩展程序选项"
   - 粘贴 API Key 并保存

4. 开始使用
   - 在任意网页选中文本
   - 点击悬浮工具栏的按钮即可翻译或朗读

================================================

⚠️ 重要提示：
- DeepSeek API 需要自行注册和充值
- 首次使用请确保网络连接正常
- 如有问题请查看 README.md 中的常见问题解答

💡 功能说明：
- 🌐 翻译：智能检测语言并翻译（中↔英）
- 🔄 语言切换：点击蓝色按钮可手动切换翻译目标
- 🔊 朗读：使用浏览器内置语音朗读文本
